**As a** bum user mid-session, **I want to** switch models freely and get a clear fail-closed login path when a provider is missing, **so that** I never depend on silent mid-turn 401 as the primary UX and never restart the CLI to change models.

Complete Login now recovery: stash deferred_model_switch, provider-scoped login, and AuthComplete auto-retry of the pending switch (per D-03). Closes MOD-06 recovery loop and reuses MOD-03 success confirmation.

Purpose: User logs in for the missing provider without re-picking the model.
Output: Deferred stash + auth complete retry GREEN with unit tests (fixture/sim AuthComplete — no live OAuth required).
</objective>
