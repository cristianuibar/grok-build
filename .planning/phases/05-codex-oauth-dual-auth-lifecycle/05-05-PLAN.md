**Discretion locks (review-mandated redesign):**
1. **Production hook (mandatory):** Call async `ensure_fresh_codex_auth` from `reconstruct_full_config()` in `session/acp_session_impl/sampler_turn.rs` for every Codex request (when `model_facts.provider == Some(Codex)`). ModelsManager prepare/snapshot may still seed chat-state credentials but **must not** be the only refresh site — chat-state stamps go stale mid-session (sampler_turn ~278-282 currently snapshot-only for Codex).
2. **Lock span:** Under in-process refresh mutex, acquire `auth.json.lock`; re-read `providers.codex`; adopt sibling rotated token if fresher; keep live guard across IdP refresh; persist with guard-held `mutate_provider_store_or_prune_with_lock(AuthProvider::Codex, …)` — mirror AuthManager refresh at manager.rs ~1618-1651 / 1693-1725.
3. **Return type:** `CodexAuthMaterial { bearer: String, account_id: Option<String> }` returned into reconstruct for `api_key` + header inject — do not rely only on mtime cache invalidation of prepare-time stamps.
4. **CodexRefresher purity:** Implements `TokenRefresher`; returns `RefreshOutcome` only; **never** mutates store (Addresses review: trait contract at refresh/mod.rs ~86-143). Storage ops live only in ensure_fresh / refresh_chain-equivalent outer function.
5. **Near-expiry policy:** JWT exp or expires_at ≤ now+5m triggers refresh when refresh_token present; fresh tokens make zero IdP calls.
6. **Transient failure:** hard-unexpired access → return old material; hard-expired → return None (no credential); never clear either provider on transient errors.
7. **Permanent failure:** clear/mark Codex only via clear_provider_slot; never AuthManager::clear; never touch xAI.
8. **Account header:** inject `ChatGPT-Account-ID` only when routed provider is Codex, credential is ChatGPT OAuth material, and endpoint is trusted ChatGPT Codex backend — **not** via global auth read inside bare inject_url_derived_headers for arbitrary hosts.
9. Injectable HTTP / TokenRefresher for tests; public or test-facing ensure_fresh entry so integration binary can prove reconstruct path without live IdP.
