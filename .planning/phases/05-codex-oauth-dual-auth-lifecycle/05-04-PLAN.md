## UI-SPEC copy locks (must match)

**Bare logout body:**
```
Specify a provider or clear all:
  bum logout --provider xai
  bum logout --provider codex
  bum logout --all
```

**Success selective:** `Logged out of {ProviderLabel}` (+ email when known)  
**Success --all:** `Logged out of all providers` (optionally list which)  
**Empty selective:** `No cached {ProviderLabel} session to log out of.`  
**Empty --all:** `No cached sessions to log out of.`  
**Status:** UI-SPEC key:value blocks; order xAI then Codex.

**TUI /logout:** Prefer description `Log out of a provider (CLI: bum logout --provider …)`; if bare `/logout` would dual-wipe, fail-closed message instead of wiping both.

**Local clear only:** no remote oauth/revoke (COVERAGE OPT-OUT).
